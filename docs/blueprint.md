# **App Name**: MitraAI

## Core Features:

- Multilingual Empathetic Chat: Chat with an AI companion that responds in Hindi or other regional languages with an empathetic tone. User can communicate through text or voice.
- Daily Mood Journal: Use a daily mood journal where the AI companion uses sentiment analysis as a tool to predict the userâ€™s mood and save it locally.
- User Interface: Provide a simple UI for users to record daily moods and view chat history.
- Anonymous Authentication: Utilize anonymous login.
- Daily Mood Reminder: Implement daily FCM (Firebase Cloud Messaging) notifications as reminders for users to share their mood.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to promote calmness and well-being.
- Background color: Light grey (#F5F5F5) to provide a gentle and neutral backdrop.
- Accent color: Muted teal (#80CBC4) for a touch of serenity and clarity.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible readability.
- Use simple, friendly icons to represent different moods and journal entries.
- Design a clean and intuitive layout for ease of navigation and user comfort.
- Implement subtle animations and transitions to enhance user engagement and a sense of calm.
